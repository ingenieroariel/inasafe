class safe.storage.projection.Projection(p)[source]¶

get_projection(proj4=False)[source]¶

Return projection

Input
proj4: If True, projection will be returned in proj4 format.
If False (default) projection will be returned in WKT format

Note: To compare projections, use the __eq__ method directly on the projection objects: E.g. self.projection == other.projection
